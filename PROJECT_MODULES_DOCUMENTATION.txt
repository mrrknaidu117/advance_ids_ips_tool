3. MACHINE LEARNING ALGORITHMS
==========================================================================

3.1 INTRUSION DETECTION MODEL
------------------------------
Files: model/model.pkl, model/scaler.pkl, train_model.py

Algorithm: Random Forest Classifier (Ensemble Learning)

Key Technologies:
• Scikit-learn - Machine learning framework
• Joblib - Model serialization and loading
• Pandas - Data processing and feature engineering
• NumPy - Numerical computations

Model Features (83 total features):
- Network flow characteristics (duration, packet counts, bytes)
- Statistical features (mean, std, min, max of packet sizes)
- Protocol-specific features (TCP flags, header lengths)
- Flow timing features (Inter-Arrival Times - IAT)
- Bidirectional flow analysis (forward/backward statistics)

Training Process:
1. Data preprocessing and feature engineering
2. Feature scaling using StandardScaler
3. Model training with cross-validation
4. Hyperparameter tuning for optimal performance
5. Model evaluation and performance metrics

Performance Metrics:
- Accuracy: ~95%+ on test datasets
- Low false positive rate for production use
- Real-time prediction capability
- Robust against various attack types

3.2 FEATURE EXTRACTION ENGINE
------------------------------
Function: extract_packet_features() in backend/sniffer.py

Purpose: Convert raw network packets into ML-ready feature vectors

Key Processing:
- Real-time packet analysis
- Statistical computation over sliding windows
- Protocol-specific feature extraction
- Flow-based aggregation
- Temporal feature calculation
